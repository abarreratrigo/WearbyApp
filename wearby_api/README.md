# Wearby
Final Proyect of my Vocational Training 

This project was developed as my final-year project for my vocational training in Cross-Platform Application Development. 

The project is an app with desktop and mobile versions. The app acts as a ‘digital wardrobe’ where users can store their clothes, add 
images, sort items using filters, and receive outfit recommendations.

IntelliJ IDEA was used as the development environment for the desktop application. JavaFX was used as the primary development language. 
The database was built using MySQL, but it has been connected to an API created with Spring Boot to allow access to the data from the 
mobile application without having to migrate the database. 

For the mobile version, Android Studio was used as the development environment. Kotlin was used as the programming language. Certain 
mobile device permissions were required to allow access to the image gallery.

Este proyecto se ha desarrollado como trabajo de final de grado de mi formación profesional en Desarrollo de Aplicaciones Multiplataforma. 

El proyecto es una aplicación con versiones de escritorio y móvil. La aplicación actúa como "armario digital" en el que el usuario puede 
almacenar sus prendas, añadir imágenes, ordenar por filtros y recibir recomendaciones de outfits.

Para el desarrollo de la aplicación de escritorio se ha usado IntellijIdea como entorno de desarollo. Para esta se ha usado JavaFX como 
lenguaje de desarrollo principal. La base de datos se ha hecho con mySQL, pero se ha conectado a una API creada con SpringBoot para poder
acceder a los datos desde la aplicación móvil sin tener que migrar la base de datos. 

Para la versión móvil se ha usado Android Studio como entorno de desarrollo. Como lenguaje se ha usado Kotlin. Se han requerido algunos 
permisos del dispositivo móvil que permiten acceder a la galería de imágenes.
